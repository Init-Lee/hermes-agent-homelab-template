# Hermes Setup Recommendations

## 推荐值

| 项目 | 推荐 |
|---|---|
| Credential rotation | Fill-first / sticky |
| Terminal backend | Local |
| Tool progress mode | all |
| Max iterations | 90 |
| Context compression threshold | 0.70–0.75 |
| Context reset | Inactivity + daily reset |
| Messaging platforms | 初期跳过 |
| Browser provider | Local Browser |
| Search provider | 初期跳过 |
| Image Generation | 初期跳过 |
| TTS | 默认免费免 key 方案 |

## 工具建议

初期保留：

- Browser Automation
- Terminal & Processes
- File Operations
- Code Execution
- Text-to-Speech
- Skills
- Task Planning
- Memory
- Session Search
- Clarifying Questions
- Task Delegation

初期暂缓：

- Web Search & Scraping
- Image Generation
- Video Generation
- Cross-Platform Messaging
- Home Assistant
- Computer Use
- Cron Jobs

## 原则

先让 CLI 和本地工作区稳定，再逐步启用高级能力。
